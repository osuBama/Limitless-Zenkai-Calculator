# Limitless-Zenkai-Calculator


Created from the spreadsheet actively in use by the community.
Uses the same logic to calculate best zenkai PLs based on record PL. Features a filter to avoid unrealistically low zenkai PLs despite better Special Power Gains.
Also contains earliest instances of PP stages to Zenkai at.

Download the HTML and open with any browser. .js file is only for website embedding, if you just want to use the calculator, you only need the HTML file.
